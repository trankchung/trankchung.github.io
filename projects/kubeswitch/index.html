Kubeswitch

I wrote this tool about a year ago to help me interact with different Kubernetes clusters. This tool allows me to switch Kubernetes context or namespace with ease. Check it out via https://github.com/trankchung/kubeswitch.

Combine this tool with tmux and you’ll be able to manage many Kubernetes clusters using just a single terminal window.
